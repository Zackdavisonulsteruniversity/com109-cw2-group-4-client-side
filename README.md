# com109-cw2-group-4-client-side

Ulster University Gym sign-up template (HTML + Bootstrap + jQuery + JavaScript + localStorage).

## Pages
- `index.html` - home page with quick interest form
- `information.html` - gym access, machines, costs, and memberships template
- `signup.html` - account creation form
- `login.html` - login form
- `dashboard.html` - protected member dashboard template

## Shared components
- Shared navbar/footer are rendered by `assets/js/main.js`
